# MeshToy

## Description
You can increase and move vertices of the mesh based on cube.
#### Video
https://twitter.com/ponsuke154/status/1066331591067099137
### Functions
* Deform the mesh by moving the sphere generated at the vertex of the mesh.
* Switch vertex, edge and face manipulation.
* Increase the number of faces while manipulating the face.
## Usage
* Create an empty object and attach a MeshGenerator to that object.
